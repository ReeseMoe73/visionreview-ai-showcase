# VisionReview AI — Public Showcase Release Post

## LinkedIn / Portfolio Announcement

I’ve published the public showcase for VisionReview AI, an AI-assisted video review and editorial workflow project.

VisionReview AI explores how AI can support editors and creators by helping move from raw footage to reviewed selects, reel drafts, QC notes, and export-ready handoff packages.

Core workflow:

```text
Upload → Analyze → Review → Build Reel → QC → Export
```

The local beta combines:

```text
FastAPI, SQLite, SQLAlchemy, OpenCV, FFmpeg, Whisper, React, and Vite
```

The public repository includes screenshots, architecture notes, sample schemas, roadmap documentation, and a product walkthrough.

Public showcase:

```text
https://github.com/ReeseMoe73/visionreview-ai-showcase
```

The full working source code remains private while I evaluate the project’s commercial direction.

This project brings together my background in AI/ML, software engineering, video production, and editorial workflow design.
