# VisionReview AI — Portfolio Demo Blurb

VisionReview AI is a local beta full-stack application for AI-assisted video review, editorial decision-making, export quality control, and client handoff package generation.

The app analyzes uploaded footage into structured segments, supports human review decisions, captures notes and tags, builds reel candidates, runs QC checks, and generates client-facing handoff deliverables.

Built with:

```text
FastAPI
SQLite
SQLAlchemy
React
Vite
OpenCV
FFmpeg
Whisper
```

The project demonstrates applied AI/ML product thinking across computer vision, transcription, full-stack development, editorial workflow design, and client delivery systems.
