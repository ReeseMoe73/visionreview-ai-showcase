- Frontend is running.
- Browser console has no red errors.
- Test video is uploaded or ready to upload.

- Project package export works.
- Notifications are turned off.

# 3–5 Minute Public Demo Script

## 0:00–0:20 — Opening

Say:

> This is VisionReview AI, a local AI-assisted video review and editorial workflow app. It helps move from raw footage to reviewed selects, reel drafts, QC notes, and export-ready handoff packages.

Show:

- Dashboard
- Local Production Workspace
- Recent Videos

---

## 0:20–0:50 — Problem

Say:

> Editors and content creators spend time scrubbing footage, finding strong moments, writing notes, organizing selects, and preparing client or editor handoff materials. VisionReview AI brings those steps into one guided workflow.

Show:

- Dashboard
- Search/filter/sort controls

---

## 0:50–1:30 — Upload and Report

Say:

> I can upload an MP4 file, and the backend analyzes the video, creates segments, generates transcripts, scores highlights, and returns an editorial report.

Show:

- Upload or existing report
- Main player
- Segment map
- Report summary
- Segment Browser

---

## 1:30–2:15 — Review Workflow

Say:

> The AI assists the process, but the editor stays in control. I can mark clips as Favorite, Keep, Reject, or Unreviewed, then add notes and tags.

Show:

- Segment review controls
- Notes
- Tags
- Review Mode briefly

---

## 2:15–3:20 — Reel Builder

Say:

> The Reel Builder turns reviewed clips into structured draft options. I can plan the draft, adjust the order, choose delivery presets, prepare client review notes, compare candidates, and run QC.

Show:

- Plan
- Order
- Presets
- Client
- QC

---

## 3:20–4:15 — Export

Say:

> Once the draft is ready, I can export clips, reels, review files, or a full project package with report data, CSV review notes, transcript, and media inventory.

Show:

- Export buttons
- Download Full Project Package

---

## 4:15–5:00 — Closing

Say:

> VisionReview AI combines applied AI, video processing, transcription, full-stack development, and editorial workflow design. This is currently a local beta, with the full source private while the project is evaluated for commercial development.

Show:

- Public showcase repo or dashboard
