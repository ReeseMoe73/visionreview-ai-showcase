# VisionReview AI — LinkedIn Launch Post

I recently completed a working local beta of VisionReview AI, an AI-assisted video review and client handoff workflow.

Demo:
https://drive.google.com/file/d/1FSOujpqpnOwJ49ciRKDRFNj_mfgv3YgT/view?usp=sharing

VisionReview AI is designed to help video teams move from raw footage to reviewed selects, draft structure, QC notes, and client-ready deliverables faster.

The current local beta includes:

- Segment-level video analysis
- Human-in-the-loop review decisions
- Notes and editorial tags
- Reel Builder workflow
- Timeline and compare views
- Export Quality Control
- Client/project metadata
- Client review HTML
- Client package ZIP export
- Send-ready handoff summaries

Most AI video tools focus on generating clips. VisionReview AI focuses on the workflow around the clips: reviewing, tagging, organizing, quality-checking, and packaging footage for client delivery.

This project combines my background in AI/ML, software development, and video production.

Built with FastAPI, SQLite, SQLAlchemy, React, Vite, OpenCV, FFmpeg, and Whisper.
