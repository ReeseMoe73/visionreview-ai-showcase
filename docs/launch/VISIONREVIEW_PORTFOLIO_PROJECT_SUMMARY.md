# VisionReview AI — Portfolio Project Summary

## Project Title

VisionReview AI — AI-Assisted Editorial Review and Client Handoff Workflow

## One-Sentence Summary

VisionReview AI is a local beta full-stack application that helps video teams move from raw footage to reviewed selects, draft structure, QC notes, and client-ready handoff packages.

## Portfolio Description

VisionReview AI analyzes uploaded footage into structured segments, supports human-in-the-loop review decisions, captures notes and editorial tags, helps organize reel candidates, runs export quality control, and generates client-facing handoff deliverables.

## Core Features

- Video upload and metadata extraction
- Segment-level video review
- Transcript-assisted review
- Highlight scoring
- Review status: Favorite, Keep, Reject, Unreviewed
- Notes and tags
- Reel Builder
- Timeline and compare workflows
- Export Quality Control
- Client/project metadata persistence
- Client review HTML export
- Client package ZIP export
- Send-ready client handoff summary

## Tech Stack

Backend: FastAPI  
Database: SQLite + SQLAlchemy  
Frontend: React + Vite  
Video processing: OpenCV  
Transcription: Whisper  
Media utilities: FFmpeg
