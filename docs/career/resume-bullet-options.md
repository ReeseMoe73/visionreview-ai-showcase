- Designed backend APIs for video upload, structured report generation, review persistence, clip/reel export, and project package delivery.

- Created a human-in-the-loop review workflow allowing users to mark clips as Favorite, Keep, Reject, or Unreviewed while adding notes, tags, and export-ready handoff data.

- Designed a human-in-the-loop review process that combines AI-derived scores with manual review decisions, tags, notes, and QC approval.
