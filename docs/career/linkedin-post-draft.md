## Longer Version

I built VisionReview AI as a local AI-assisted video review and editorial workflow application.

The goal is to help editors, creators, and production teams reduce the time spent scrubbing footage, identifying strong moments, organizing selects, writing notes, preparing client summaries, and exporting handoff materials.

VisionReview AI supports:

- MP4 upload
- Segment-level video analysis
- Transcript generation
- Highlight scoring
- Human review decisions
- Review notes and tags
- Reel Builder workflow
- Client review summaries
- QC and approval checks
- Project package exports
- Recent project dashboard

The current build is a Local Beta v1.0. The public GitHub repository is a showcase version with screenshots, architecture notes, sample schemas, and product documentation. The full working source code remains private while I evaluate future commercial possibilities.

Public showcase:

```text
https://github.com/ReeseMoe73/visionreview-ai-showcase
```
