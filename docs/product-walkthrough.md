# VisionReview AI — Product Walkthrough

## Overview

VisionReview AI helps turn uploaded footage into reviewed, organized, and export-ready editorial drafts.

```text
Upload → Analyze → Review → Build → QC → Export
```

## 1. Upload

A user uploads an MP4 file into the local application.

## 2. Analyze

The system extracts metadata, creates segments, generates thumbnails, scores quality, and produces transcript information.

## 3. Review

The user marks segments as Favorite, Keep, Reject, or Unreviewed, then adds notes and tags.

## 4. Build Reel

The Reel Builder helps organize selected clips into possible drafts using Plan, Order, Presets, Client, Timeline, Copy, Handoff, Compare, and QC.

## 5. QC and Approval

The QC workflow checks runtime, clip count, hook, CTA, visual warnings, review completeness, and approval notes.

## 6. Export

The user can export clips, reels, review files, and a project package.
