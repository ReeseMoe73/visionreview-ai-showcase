```text

```

Why:

- Useful for SkreenKings
- Client-facing
- Does not require public SaaS

## What Not to Build Yet

Avoid these for now:

- Full SaaS auth system
- Payment system
- Multi-user permissions
- Cloud storage
- Advanced admin dashboard
- Complex AI model training
- Marketplace features

These are only worth building after demand is validated.
