### Description

```text
VisionReview AI is a Local Beta v1.0 full-stack AI/video workflow application designed to help editors, creators, and production teams move from raw footage to reviewed clips, structured reel drafts, QC notes, client summaries, and export-ready project packages.

The system uses a FastAPI backend, React/Vite frontend, SQLite/SQLAlchemy persistence, OpenCV and FFmpeg media processing, and Whisper-based transcription workflows. The app includes upload, segmentation, transcript generation, highlight scoring, human review decisions, Review Mode, Reel Builder, client review summaries, QC approval, and export package workflows.

The full working source code remains private while the project is evaluated for commercial development. The public GitHub repository is a showcase version with screenshots, architecture notes, sample schema, roadmap, and portfolio documentation.
```
