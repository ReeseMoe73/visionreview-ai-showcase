# VisionReview AI — Resume Project Section

## Recommended Resume Project Entry

### VisionReview AI — AI-Assisted Video Review & Editorial Workflow System  
**Full-Stack AI / Video Workflow Application | Local Beta v1.0**  
**Tech:** FastAPI, React, Vite, SQLite, SQLAlchemy, OpenCV, FFmpeg, Whisper, JavaScript, Python

Built a local full-stack application that helps transform uploaded footage into reviewed clips, structured reel drafts, QC notes, and export-ready editorial handoff packages. Designed the system around a human-in-the-loop workflow where AI-assisted analysis supports, but does not replace, editorial judgment.

- Developed a FastAPI backend for MP4 upload, metadata extraction, segment generation, transcript processing, structured report creation, review persistence, clip/reel export, and project package delivery.
- Built a React/Vite frontend with dashboard search/filter/sort, segment review controls, Review Mode, Reel Builder, client review summaries, QC approval workflow, and export actions.
- Integrated OpenCV, FFmpeg, and Whisper-based processing to support video metadata extraction, media segmentation, audio/transcription workflow, and editorial analysis.
- Designed human-in-the-loop review features allowing users to mark segments as Favorite, Keep, Reject, or Unreviewed while adding review notes, tags, quick actions, and approval notes.
- Created Reel Builder workflows for draft planning, clip ordering, platform presets, client review, timeline notes, posting copy, editor handoff, candidate comparison, and QC readiness checks.
- Packaged the project as a Local Beta v1.0 release with private source-code management and a public showcase repository containing documentation, screenshots, sample schema, roadmap, and demo planning materials.
