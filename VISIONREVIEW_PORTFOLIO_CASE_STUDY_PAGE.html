Portfolio Case Study

VisionReview AI

An AI-assisted video review and editorial workflow application for moving from raw footage to reviewed selects, reel drafts, QC notes, and export-ready handoff packages.

FastAPIReactOpenCVFFmpegWhisperSQLite

Project Snapshot

Project TypeFull-stack AI/video workflow application
StatusLocal Beta v1.0
RoleFull-stack developer and AI workflow designer
FocusApplied AI, video analysis, editorial workflow

The Problem

Editors and creators spend significant time scrubbing footage, finding strong moments, organizing selects, writing notes, preparing client summaries, and exporting handoff materials across disconnected tools.

The Solution

VisionReview AI brings the process into one guided workflow:

Upload → Analyze → Review → Build Reel → QC → Approve → Export

Key Features

DashboardUpload, reopen, search, filter, sort, and manage local projects.
Review WorkflowMark clips as Favorite, Keep, Reject, or Unreviewed with notes and tags.
Reel BuilderPlan, order, preset, client review, timeline, copy, handoff, compare, and QC tools.
ExportsClip, reel, JSON, CSV, TXT, and Project Package ZIP exports.
